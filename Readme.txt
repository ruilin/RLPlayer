本程序是安卓平台下最简单的基于FFmpeg的视频解码器。它可以将输入的视频数据解码成YUV像素数据。本程序中将FFmpeg所有的类库打包成为了一个类库。

This software is the simplest decoder based on FFmpeg in Android. It can decode video stream
to raw YUV data. In this software we packed all FFmpeg librarys to one library.
